# Trabajo Práctico - Portafolio Front End

Este proyecto es un **Trabajo Práctico (TP)** de Desarrollo Front End. Consiste en la creación de un portafolio personal utilizando HTML y CSS, con estructura semántica, diseño responsivo y publicación en GitHub Pages.

## Checklist de Requerimientos

- [x] Archivo `index.html` en la raíz del proyecto
- [x] Carpeta `css` con `styles.css`
- [x] Carpeta `img` opcional para recursos gráficos
- [x] Archivo `README.md` con descripción y checklist
- [x] Repositorio en GitHub
- [x] Publicación en GitHub Pages
- [x] Uso de Google Fonts
- [x] DOCTYPE y metaetiquetas correctas
- [x] Header con título
- [x] Nav con 3 enlaces
- [x] Main con secciones: sobre mí, tarjetas, habilidades, contacto, películas
- [x] Footer con información adicional
- [x] Comentarios en el código
- [x] CSS con selectores, tipografía, Flexbox/Grid, responsivo
- [x] Animaciones/transiciones en tarjetas y botones
